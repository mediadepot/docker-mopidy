# Requirements


# Environmental
The following environmental variables must be populated, when running container 

- PUID
- PGID


# Ports
The following ports must be mapped, when running container 

 - 8081 #webui listen 
 
# Volumes
The following volumes must be mapped, when running container 

- :/srv/mopidy/config
- :/srv/mopidy/data
- /mnt/music:/mnt/music
